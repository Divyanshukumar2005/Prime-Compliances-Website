# Prime Compliances — Business Website

A responsive static website for **Prime Compliances**, a business compliance and company registration services provider. Built with vanilla HTML, CSS, and JavaScript.

## Features

- Fully responsive, mobile-first design
- Service sections for business compliance & registration offerings
- WhatsApp-integrated enquiry/contact forms
- Testimonials section
- SEO-friendly (sitemap.xml, robots.txt)

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Getting Started

Clone the repo and open `index.html` in your browser — no build step required.

```bash
git clone https://github.com/Divyanshukumar2005/Prime-Compliances-Website.git
cd Prime-Compliances-Website
```
