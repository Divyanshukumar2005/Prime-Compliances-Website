## 📌 About the Project

**Prime Compliances** is a business compliance and company registration services provider helping startups and businesses in India navigate legal formalities — from company incorporation to regulatory filings.

This repository contains the **official marketing website**, built to:

- Establish a credible, professional online presence for the brand
- Convert visitors into leads through a frictionless **WhatsApp-based enquiry flow**
- Clearly present the company's service catalog and social proof (testimonials)
- Rank well on search engines through on-page SEO best practices

> Designed and developed independently as a **freelance client project**, covering everything from UI/UX design to deployment.

## 🚀 Getting Started

No build tools, no dependencies — just clone and open.

```bash
# Clone the repository
git clone https://github.com/Divyanshukumar2005/Prime-Compliances-Website.git

# Move into the project directory
cd Prime-Compliances-Website

# Open in your default browser
open index.html      # macOS
# or simply double-click index.html
```
